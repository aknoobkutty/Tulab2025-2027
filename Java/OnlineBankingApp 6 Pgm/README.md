# OnlineBankingApp

" 6 . A simple online banking demo using JSP and session data. "

## Features
- Login with a demo user account
- View account balance
- Deposit funds
- Withdraw funds
- Logout

## Demo credentials
- Username: `customer1`
- Password: `bank123`

## Run instructions
1. Deploy the `OnlineBankingApp` folder to a JSP/Servlet container such as Apache Tomcat.
2. Ensure the container uses Java and JSP support.
3. Access the app at `http://localhost:8080/OnlineBankingApp/`.

## Files
- `index.jsp` - login page
- `dashboard.jsp` - account dashboard and transaction forms
- `process.jsp` - handles login, deposit, and withdrawal actions
- `logout.jsp` - ends the user session
- `WEB-INF/web.xml` - web application descriptor
